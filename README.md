# ClaudeConnection

> A sleek, animated landing experience powered by Claude AI.

## ✦ Overview

**ClaudeConnection** is a beautifully crafted web interface that showcases the power of Claude AI through an immersive, animated HTML/CSS/JS frontend. Designed for impact — refined for real use.

## ✦ Features

- 🌌 **Immersive animated background** — layered particle effects and flowing gradients
- ✨ **Smooth micro-interactions** — every hover, click, and transition is intentional
- 📱 **Fully responsive** — looks stunning on all screen sizes
- ⚡ **Zero dependencies** — pure HTML, CSS, and vanilla JS
- 🎨 **Production-grade design** — editorial typography, cohesive palette, refined spacing

## ✦ Getting Started

```bash
# Clone the repo
git clone https://github.com/your-username/ClaudeConnection.git

# Navigate to the project
cd ClaudeConnection

# Open in browser
open index.html
```

No build step. No bundler. Just open `index.html` and it runs.

## ✦ Customization

All design tokens live as CSS custom properties at the top of `index.html`:

```css
:root {
  --ink:        #0a0a0f;
  --paper:      #f5f0e8;
  --accent:     #e8460a;
  --accent-alt: #f5c842;
  --serif:      'Playfair Display', serif;
  --mono:       'JetBrains Mono', monospace;
}
```

Swap them to instantly retheme the entire interface.
